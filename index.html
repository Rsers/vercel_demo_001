🚀 Vercel API 演示

这个页面展示了如何使用 Vercel 的 Serverless Functions 创建 API 接口。

GET/api/hello

简单的问候 API，支持查询参数。

// 请求示例 fetch('/api/hello?name=张三') .then(response => response.json()) .then(data => console.log(data));

POST/api/users

创建用户数据的 API。

// 请求示例 fetch('/api/users', { method: 'POST', headers: { 'Content-Type': 'application/json' }, body: JSON.stringify({ name: '张三', email: 'zhangsan@example.com', message: '你好世界' }) }) .then(response => response.json()) .then(data => console.log(data));

GETPOSTPUTDELETE/api/tasks

任务管理 API，支持所有 HTTP 方法。

// GET - 获取所有任务 fetch('/api/tasks') // POST - 创建任务 fetch('/api/tasks', { method: 'POST', headers: { 'Content-Type': 'application/json' }, body: JSON.stringify({ title: '新任务', description: '任务描述' }) }) // PUT - 更新任务 fetch('/api/tasks?id=1', { method: 'PUT', headers: { 'Content-Type': 'application/json' }, body: JSON.stringify({ title: '更新的任务', completed: true }) }) // DELETE - 删除任务 fetch('/api/tasks?id=1', { method: 'DELETE' })
